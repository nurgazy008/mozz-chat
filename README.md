# Mozz Messenger

Мессенджер на Flutter с Firebase Firestore.

## Функционал

- Список чатов с поиском
- Экран чата с сообщениями
- Отправка сообщений в реальном времени
- Форматирование дат и времени
- Аватары с инициалами
- Статус "В сети"

## Запуск

```bash
flutter pub get
flutter run
```

## Сборка APK

```bash
flutter build apk --release
```
